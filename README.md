## 使用说明
1. docker-compose 快速启动
```shell script
# 拉取代码
cd luban
docker-compose up -d
```

2. 启动服务前先修改etc/config.yaml

3. 导入初始化sql, 并逐步升级到最新版本
```shell
全量更新SQL: https://docs.dnsjia.com/upgrade/sql/v2.4.0.sql
升级文档: https://docs.dnsjia.com/upgrade/changelog/
```

4. 初始账号: admin  密码: luban123.
